# Project Name

Phoenix Pages

<br>

## Description

Online personal library that helps you track your reading journey.

<br>

## Data

User

```json
{
  "username": String,
  "password": String,
  "id": Number
}

```

Books To Read

```json
{
  "id": Number,
  "bookKey": String,
  "userId": Number
}

```

Books Already Read

```json
{
  "id": Number,
  "bookKey": String,
  "userId": Number
}

```
